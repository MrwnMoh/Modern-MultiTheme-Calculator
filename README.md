## 📝 Introduction
**Modern Multi-Theme Calculator** is a professional desktop application developed using **C# WinForms**. This project focuses on bridging the gap between complex backend logic and a high-quality user interface, featuring a dynamic theme engine and optimized calculation processing.

## 🚀 Main Features

### 🎨 1. Dynamic Multi-Theme UI
* **Dark Mode:** A sleek, high-contrast interface designed for professional use.
* **Nature & Light Themes:** Additional modes switchable via a **Recursive Theme Engine** that updates all controls instantly.
* **Flat Design:** Modern button styling with no borders for a contemporary feel.

### 🧮 2. Smart Calculation Engine
* **DataTable.Compute:** Uses a robust engine to handle mathematical expressions as dynamic strings.
* **Advanced Modulus:** Includes a dedicated `%` (Remainder) operator for programming logic.
* **Calculation Tracker:** A real-time history display to track your current operation.

### 🛠️ 3. Optimized Code Structure
* **Recursive Logic:** Implemented a recursive loop to traverse control containers and update styles.
* **Input Validation:** Prevents common errors like multiple decimal points or invalid operations.
* **Clean Events:** Centralized event handling for cleaner and more maintainable code.
